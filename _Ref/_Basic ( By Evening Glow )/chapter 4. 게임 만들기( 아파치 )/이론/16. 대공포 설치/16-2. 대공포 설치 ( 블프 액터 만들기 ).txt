0.	새 클래스 블프 만들기
-------------------------------------

	-	[ 블프 / 새 클래스 블프 / 액터 ]로 실행 후

		이름은 [ BP_Turret ]로 설정.


-------------------------------------
1.	트럭 & 포탑 추가
-------------------------------------

	-	[ 콘텐츠 브라우저 ]에서 트럭_DM, 포탑_DM을
	
		차례로 [ 컴포넌트 ]탭으로 드래그 한다.

		스태틱 메시의 피벗을 맞춘 상태로 디자인 되어 있으므로

		자동으로 조립됨.

	-	컴포넌트 추가후 해당 컴포넌트가 x축( 전방 )을 향하고 있는지 확인.



-------------------------------------
2.	Spawn Point 설치
-------------------------------------

	-	포탑에 로켓을 발사할 발사위치를 설정한다.

	-	Arrow를 설치한후 다음과 같이 설정한다.

	-	대공포는 Pawn이 아니므로 로켓과 포신의 충돌을

		자동으로 방지하는 방법은 없다.

		충돌이벤트에서 판정 할 수 있지만 절차가 번거로우므로

		컴포넌트 구성시 미리 충돌을 방지 한다.


			카테고리		|	속성		|	값								|	비고
		-------------------------------------------------------------------------------------------
			변수			|	변수 이름	|	SpawnRight	/	SpawnLeft		|	오른쪽 / 왼쪽 발사위치
		-------------------------------------------------------------------------------------------
			트랜스폼		|	위치		|	(220, 57, 7) / (220, -57, 7)	|	화살표 위치
							|	크기		|	(0.5, 1, 1)						|	화살표 크기
		-------------------------------------------------------------------------------------------
			Arrow Component	|	Arrow Color	|	적당한 색						|	화살표 색
		-------------------------------------------------------------------------------------------
			기타			|	포탑 회전시 함께 회전해야 하므로 포탑 컴포넌트의 하위개체로 만듬.
		-------------------------------------------------------------------------------------------


-------------------------------------
3.	Hit 이벤트 속성 설정
	( 4.2.1 버전 기준, 현재 버전에서는 해당 사항이 발생하지 않음.)
-------------------------------------

	-	현재 트럭이 먼저 파괴된 경우 포탑이 공중에 떠 있는 상태가 된다.

		잠깐이지만 이 상태에서 대공포를 발사하면 비현실적이다.

		->	로켓이 트럭에 명중하더라도 포탑이 함께 파괴되는 기능을 추가한다.

	-	트럭의 [ Collision / Simulation Generates Hit Event ]를 On 하면

		로켓이 트럭에 명중할 때 Hit 이벤트 발생.

		이때 물리처리를 진행하므로 트럭의 파편이 포탑을 파괴.

		추후 Hit 이벤트는 부서진 대공포의 파편을 제거하는데 사용.



-------------------------------------
4.	대공포의 헬기 탐지.
-------------------------------------

	-	대공포는 포탑과 트럭으로 구성되어 있지만,

		헬기를 탐지하고 로켓을 발사하는 컴포넌트는 포탑이다.

	
	-	포탑이 헬기를 탐색하는 프로세스.

		1)	대공포의 위치를 파악.

			->	Get Actor Location.

		2)	헬기의 위치를 파악.

			->	Get Player Pawn - Get Actor Location.

		3)	포탑 -> 헬기의 방향과 거리 파악.

			->	Find Look at Rotation.

		4)	3)의 방향으로 포탑 회전.

			->	Gun - Set World Rotation.

	-	블프 작성.

		-	실시간 탐지를 해야하므로 [ Tick 이벤트 ]에서 처리한다.

		-	게임 실행후 플레이어 이동시 대공포가 모두 파괴된다 !!!!!

			->	포탑과 트럭의 콜리전이 서로 충돌하기 때문.

				->	포탑과 트럭의 간격을 약간 띄운다.
