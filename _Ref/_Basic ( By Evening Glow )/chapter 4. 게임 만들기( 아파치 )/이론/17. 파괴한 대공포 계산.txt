========================================================
@	파괴한 대공포 계산
========================================================

-------------------------------------
0.	개요
-------------------------------------

	-	파괴한 대공포의 수를 화면에 표시.

	-	충돌의 판정과 처리를 어디서 하느냐에 따라 블프 처리방식이 달라짐.

	-	충돌이 발생하는 경우.

		------------------------------------------
		주체		|		대상
		------------------------------------------
		헬기 로켓	|	대공포, 지면, 바위
		대공포 로켓	|	헬기, 바위, 장애물, 바위 파편
		바위 파편	|	헬기, 대공포, 로켓
		헬기		|	대공포 로켓, 지면, 바위, 바위파편
		------------------------------------------

	-	충돌체나 피충돌체 모두 충돌 속성이 설정되면 Hit 이나 Overlap 이벤트 발생.

		-	대부분의 경우 충돌을 당하는 측에서 충돌의 판정과 처리를 진행하는 것이 전체적인 구조가 간단해짐.

		-	파괴한 대공포의 수를 저장하는 변수는 헬기 블프에 추가.

			그 값을 증가시키는 것은 피격당한 대공포에서 처리.

			----------------------------------------------------------------------------------
			변수명		|	Type	|	초기값	|	편집 가능	|	설명
			----------------------------------------------------------------------------------
			KillCount	|	int		|	0		|	On			|	피격한 적의 수.
			EnemyCount	|	int		|	3		|	On			|	씬에 배치된 적의 수.
			----------------------------------------------------------------------------------


-------------------------------------
1.	KillCount 증가함수 추가.
-------------------------------------

	-	SetKillCount 함수 추가.

		-	대공포 파괴시 헬기의 블프 변수 KillCount를 1씩 증가 처리.

		-	[ BP_Turret ]에서 작성.

		-	[ GetPlayerPawn -> BP_Appache에 형변환 ]노드로 어느 블프에서든지

			헬기 블프 변수의 값을 읽고 쓸수 있다.

		-	변수는 편집 가능 속성이 설정되어 있어야 하며

			함수도 접근 지정자가 퍼블릭으로 설정되어 있어야( 함수의 기본 속성 ) 외부에서 편집이 가능.

		-	대공포는 헬기의 로켓, 바위 파편에 의해서 파괴될 수 있다.

			어떤 형태든 KillCount 값이 증가함.
		
		-	[ BP_Turret / SetKillCount ] 노드 참고.



-------------------------------------
2.	대공포 블프 수정.
-------------------------------------

	-	[ BP_Turret / OncomponentHit ]에서 SetKillCount를 호출한다.


-------------------------------------
3.	헬기 블프 수정.
-------------------------------------

	-	게임의 UI 등은 HUD( Head Up Display )시스템에서 처리한다. ( 추후 진행 )

	-	우선 PrintString으로 디버그 출력한다.

	-	[ Tick 이벤트 ]의 시퀀스 1번 참고.
