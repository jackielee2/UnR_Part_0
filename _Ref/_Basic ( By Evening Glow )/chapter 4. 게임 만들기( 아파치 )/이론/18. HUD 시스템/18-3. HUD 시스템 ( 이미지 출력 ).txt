-------------------------------------
1.	이미지 출력 노드
-------------------------------------
	
	-	HUD 에서 이미지를 출력하는 방법은 다음과 같다.

		-	함수뒤 접미사 Simple는 유사한 이름의 함수의 기능을 단순화 한 것.

		-	 종류.

			이름					|	설명
			----------------------------------------
			Draw Texture			|	2D 이미지 출력
			Draw Texture Simple		|
			----------------------------------------
			Draw Material			|	머터리얼 출력
			Draw Material Simple	|
			----------------------------------------

		-	주요 속성.

			이름						|	설명
			--------------------------------------------------------------------
			Texture						|	출력할 2D 이미지
			--------------------------------------------------------------------
			Screen (X, Y)				|	이미지의 출력 위치.
										|	픽셀단위.
										|	화면 좌상단이 (0,0)
			--------------------------------------------------------------------
			Screen (W, H)				|	이미지의 폭과 높이.
										|	픽셀단위.
										|	실제 화면에 표시되는 이미지 크기.
			--------------------------------------------------------------------
			Texture (U, V)				|	출력할 이미지의 기준 위치.
										|	이미지의 좌상단이 (0,0), 우하단이 (1,1)
			--------------------------------------------------------------------
			Texture (UWidth, VHeight)	|	이미지 반복 ( 타일링) 횟수.
										|	(1,1)은 이미지 전체 출력.
			--------------------------------------------------------------------
			TintColor					|	이미지에 덧씌울 생상.
			--------------------------------------------------------------------
			Blend Mode					|	이미지 표시 방법.
										|	투명색 적용, 완전 불투명 등 옵션있음.
			--------------------------------------------------------------------
			Scale						|	이미지 확대 비율.
			--------------------------------------------------------------------
			Scale Position				|	이미지를 확대할 때 화면 좌표(Screen(X, Y))도 확대하는지 여부.
			--------------------------------------------------------------------
			Rotation					|	이미지 회전 각도.
										|	시계방향으로 회전. 60분법 각도( Degree )사용.
			--------------------------------------------------------------------
			Rot Pivot (x, y)			|	이미지 회전 시 중심점.
										|	UV 좌표계 사용.
			--------------------------------------------------------------------





-------------------------------------
2.	UV 좌표
-------------------------------------

	-	매핑( Mapping )

		-	3D 오브젝트의 표면을 2D 이미지( 텍스쳐 )로 감싸는 것.
		
	-	오브젝트 매핑시 텍스쳐의 기준점, 그리는 방향, 텍스쳐의 반복 횟수등 필요.

		-	이와 같은 처리를 위해 오브젝트의 축( x, y, z )에 대응해서 생겨난 개념이

			2D 텍스쳐의 축을 ( u, v, w )로 정의하는 방법.

		-	2D 이미지인 텍스쳐는 2개의 축만 필요하므로 (u,v)를 사용.

	-	UV는 텍스처 내부의 위치를 표시하는 논리적인 좌표로, 텍스처의 크기와 상관없이

		이미지의 좌상단은 (0,0), 우하단은 (1,1)로 정의.

		즉 UV (0.5, 0.5)는 텍스쳐의 한가운데 지점.

	-	타일링

		-	이미지 반복.

		-	UV 값으로 매핑되는 이미지를 반복 처리 할수있음.

		-	타일링 값이 1보다 크면 이미지 반복.

						1보다 작으면 텍스쳐 일부분만 사용.

	-	언리얼의 타일링 방식.

		-	HUD의 Draw

			-	Texture UWidth, VHeight로 표시.

		-	머터리얼 에디터

			-	UTiling, VTiling으로 표시.

		


-------------------------------------
3.	Draw Material
-------------------------------------

	-	Draw Material

		-	머터리얼을 게임 화면에 표시하는 기능.

		-	모든 머터리얼을 출력 할 수 있지만 이미시브 컬러에 설정도니 값만 반영.

		-	이미시브 컬러가 없는 머터리얼은 전부 검은색으로 표현됨.
