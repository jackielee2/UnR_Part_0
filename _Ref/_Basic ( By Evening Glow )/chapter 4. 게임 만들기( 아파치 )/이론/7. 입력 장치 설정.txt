========================================================
@	입력 장치 설정.
========================================================

----------------------
0.	개요
----------------------

	*	주요 입력 장치는 키보드, 마우스, 터치패드 등.
	
	*	Action Mappings vs Axis Mappings

		Action

			-	누를 때마다 한 번씩 동작하는 입력 장치.

			-	점프, 발사 등.

		Axis

			-	누르고 있는동안 계속해서 동작하는 입력장치.

			-	이동, 회전 등.

----------------------
1.	설정.
----------------------

	1.	Action Mappings 설정

		----------------------------------------------------------------------------------------------
			매핑 이름					|		장치					|			비고
		----------------------------------------------------------------------------------------------
			Apache Shoot				|	왼쪽 ctrl					|	로켓 발사 키
										|	왼쪽 마우스 버튼			|	로켓 발사 버튼
										|	2 터치 ( 두 손가락 터치 )	|	모바일 터치.
		----------------------------------------------------------------------------------------------
			Apache first/third view		|	X							|	1/3인칭 전환 키
										|	핀치 ( 두 손가락 줌 )		|	???( 최신 버전( 4.22.3 )에는 보이지 않음 )
		----------------------------------------------------------------------------------------------
			Apache Popup Menu			|	M							|	게임 Popup 메뉴 호출 키.
										|	글로벌 뒤로					|	뒤로 가기 버튼. 게임 패드.
										|	안드로이드 뒤로				|	안드로이드 버튼.
		----------------------------------------------------------------------------------------------

	2.	Action Mappings 설정

		-----------------------------------------------------------------------------------------------------------
			매핑 이름		|		장치											|			비고
		-----------------------------------------------------------------------------------------------------------
			전후 이동		|	W / S												|	1, -1
							|	게임 패드 왼쪽 썸스틱 Y 축( 게임패드 왼쪽 Y )		|	1, 왼쪽 조이스틱의 상하 이동 값.
		-----------------------------------------------------------------------------------------------------------
			좌우 이동		|	A / D												|	-1, 1
							|	게임 패드 왼쪽 썸스틱 X 축( 게임패드 왼쪽 X )		|	1, 왼쪽 조이스틱의 좌우 이동 값.
		-----------------------------------------------------------------------------------------------------------
			상하 이동		|	Z /C												|	-1, 1
							|	게임 패드 오른쪽 썸스틱 Y 축( 게임패드 오른쪽 Y )	|	-1, 오른쪽 조이스틱의 상하 이동 값.
		-----------------------------------------------------------------------------------------------------------
			회전			|	Q / E												|	-1, 1
							|	게임 패드 오른쪽 썸스틱 Y 축( 게임패드 오른쪽 X )	|	1, 오른쪽 조이스틱의 좌우 이동 값.
		-----------------------------------------------------------------------------------------------------------


	*	입력한 매핑은 블프 액션 목록의 입력 카테고리에 표시 된다.

	*	키 매핑은 [ 프로젝트 / Config / DefaultInput.init ]파일에 저장된다.

		이 파일을 다른 프로젝트의 Config 폴더에 복사하면 같은 키 매핑을 사용할 수 있음.
